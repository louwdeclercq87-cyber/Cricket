# Cricket Simulation Game

This project is a simulation of a cricket game allowing users to play matches, manage teams, and simulate scenarios.

## Features
- Match simulation
- Team management
- Various game modes

## Installation
1. Clone the repository
2. Install dependencies
3. Run the application

## Usage
To start the game, run the following command:
```bash
python main.py
```
